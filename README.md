# RIEM Assignment 2

Repository for Assignment 2 of the DTU course **46755 Renewables in Electricity Markets**.

## Topic

**Market Participation from a Stakeholder Perspective**

The assignment is divided into two main parts:

1. **Step 1:** Participation in day-ahead and balancing markets for a price-taking wind farm.
2. **Step 2:** Participation in ancillary service markets for a stochastic flexible load providing DK2 FCR-D UP reserve.

## Repository structure

```text
data/       Input data and generated scenarios
src/        Python source code
results/    Generated figures, tables and numerical outputs
main.py     Main script to run the workflow
```

## How to run

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the main script:

```bash
python main.py
```

## Notes

The code is structured to keep the assignment workflow simple and reproducible.

Figures and tables used in the report should be generated from the scripts and saved in the `results/` folder.
